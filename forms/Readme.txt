Resume Website - Naomi Kapele
=============================

Project Overview:
This is a personal resume website showcasing my skills, experience, and portfolio projects. 
It is built using modern web technologies and is fully  responsive. 

Technologies Used:
- HTML, CSS, JavaScript
- PHP (for contact form)
- XAMPP for local development
- GitHub for version control and deployment

Features:
- Responsive personal portfolio layout
- Contact form with email integration (Gmail SMTP)
- Projects and skills showcase
- Easy to customize and deploy

Deployment:
- Can be hosted on GitHub Pages, Heroku, or any PHP-enabled server
- Contact form requires Gmail SMTP credentials for email functionality
